## What This Guide Covers

This guide provides a structured onboarding path for new team members joining a SPICE Harvester deployment, including:

- **Role-Based Learning Paths** -- Tailored onboarding tracks for data engineers, platform administrators, integration developers, and business analysts
- **Platform Overview** -- What SPICE Harvester does, who uses it, and how it fits into the data ecosystem
- **Key Terminology** -- Essential vocabulary (ontology, object types, properties, instances, actions, events)
- **First Tasks** -- Hands-on exercises for each role to build familiarity
- **Common Workflows** -- The most frequent tasks users perform and how to accomplish them
- **Getting Help** -- Where to find documentation, support channels, and escalation paths
- **Environment Access** -- Obtaining credentials, configuring development tools, and connecting to the platform
