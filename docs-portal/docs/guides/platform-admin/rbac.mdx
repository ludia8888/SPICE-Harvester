## What This Guide Covers

This guide will walk platform administrators through configuring role-based access control, including:

- **RBAC Model** -- Understanding users, roles, permissions, and policies in SPICE Harvester
- **Built-in Roles** -- Default roles (Admin, Editor, Viewer) and their permissions
- **Custom Roles** -- Creating roles with fine-grained permissions per ontology, object type, and operation
- **User Management** -- Creating, updating, and deactivating user accounts
- **Permission Scoping** -- Restricting access by ontology, object type, property, and action type
- **API Token Management** -- Creating and rotating service account tokens
- **Audit Logging** -- Reviewing access logs and permission changes
- **SSO Integration** -- Configuring SAML/OIDC for enterprise single sign-on
