## What This Guide Covers

This guide will walk platform administrators through monitoring SPICE Harvester services, including:

- **Grafana Dashboards** -- Navigating pre-built dashboards for service health, request latency, and throughput
- **Prometheus Metrics** -- Understanding the key metrics exposed by each service (request rate, error rate, latency percentiles)
- **Jaeger Tracing** -- Using distributed tracing to debug slow requests and cross-service issues
- **Log Aggregation** -- Querying logs in Loki/Grafana for troubleshooting
- **Alert Configuration** -- Setting up AlertManager rules for critical conditions (service down, high error rate, disk full)
- **Health Check Endpoints** -- Monitoring service health via HTTP endpoints and Docker health checks
- **Capacity Planning** -- Using metrics to plan scaling decisions
- **Custom Dashboards** -- Building Grafana dashboards for your specific monitoring needs
