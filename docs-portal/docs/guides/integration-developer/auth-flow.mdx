## What This Guide Covers

This guide will walk integration developers through SPICE Harvester's authentication and authorization mechanisms, including:

- **Authentication Methods** -- Bearer token authentication, refresh tokens, and service account tokens
- **Token Lifecycle** -- Obtaining, refreshing, and revoking access tokens
- **Authorization Model** -- How roles and permissions are evaluated on each request
- **Service-to-Service Auth** -- Authenticating background services and workers
- **SSO/OIDC Integration** -- Connecting to enterprise identity providers
- **Token Security** -- Best practices for storing and transmitting tokens
- **Multi-Tenant Auth** -- Per-ontology permission scoping
- **Debugging Auth Issues** -- Common authentication and authorization error codes and resolutions
