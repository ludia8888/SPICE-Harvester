## What This Guide Covers

This guide will walk business analysts through building dashboards and visualizations with SPICE Harvester data, including:

- **Dashboard Overview** -- Available dashboard tools and how they connect to SPICE Harvester data
- **Grafana Dashboards** -- Creating custom Grafana panels that query SPICE Harvester metrics and data
- **Data Exploration** -- Using the built-in web UI to explore ontology objects and relationships
- **Search-Powered Views** -- Building filtered views using SearchJsonQueryV2 queries
- **Projection-Based Analytics** -- Leveraging projections for aggregated and computed views
- **Charting and Tables** -- Configuring chart types, table columns, and formatting
- **Filtering and Drill-Down** -- Adding interactive filters to narrow down data
- **Sharing Dashboards** -- Exporting and sharing dashboard configurations
- **Embedding** -- Embedding SPICE Harvester visualizations in external applications
