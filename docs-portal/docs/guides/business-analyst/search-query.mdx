## What This Guide Covers

This guide will walk business analysts through searching and querying data in SPICE Harvester, including:

- **Search Basics** -- Using the v2 Search API to find objects by property values
- **Operator Reference** -- Working with all 13 search operators (`eq`, `gt`, `gte`, `lt`, `lte`, `isNull`, `containsAnyTerm`, `containsAllTerms`, `containsAllTermsInOrder`, `containsAllTermsInOrderPrefixLastTerm`, `startsWith`, `and`, `or`, `not`)
- **Text Search** -- Full-text search with term matching, phrase matching, and prefix matching
- **Combining Filters** -- Building complex queries with `and`, `or`, and `not` operators
- **Sorting and Pagination** -- Ordering results and navigating large result sets
- **Selective Fields** -- Using `select` to return only the properties you need
- **Search Performance** -- Tips for writing efficient queries
- **Common Query Patterns** -- Date ranges, multi-value filters, null checks, and typeahead/autocomplete
