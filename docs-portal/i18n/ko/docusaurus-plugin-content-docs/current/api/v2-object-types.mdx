# Foundry v2 객체 유형 API

객체 유형(Object Types)은 온톨로지 내 엔티티의 스키마를 정의합니다. 이 엔드포인트를 사용하여 객체 유형 정의를 조회하고 검사할 수 있습니다.

## 객체 유형 목록 조회

특정 온톨로지에 정의된 모든 객체 유형을 조회합니다.

```
GET /api/v2/ontologies/{ontologyApiName}/objectTypes
```

### 경로 파라미터

| 파라미터 | 타입 | 필수 | 설명 |
|:---------|:-----|:-----|:-----|
| `ontologyApiName` | string | 예 | 온톨로지의 API 이름 (예: `acme`) |

### 쿼리 파라미터

| 파라미터 | 타입 | 필수 | 설명 |
|:---------|:-----|:-----|:-----|
| `pageSize` | integer | 아니오 | 페이지당 결과 수 (기본값: 100, 최대: 1000) |
| `pageToken` | string | 아니오 | 다음 페이지를 위한 불투명 토큰 |

### 요청 예시

```bash
curl -X GET "http://localhost:8002/api/v2/ontologies/acme/objectTypes?pageSize=25" \
  -H "Authorization: Bearer <token>"
```


### 응답 필드

| 필드 | 타입 | 설명 |
|:-----|:-----|:-----|
| `data` | array | 객체 유형 정의 목록 |
| `data[].apiName` | string | 객체 유형의 고유 식별자 |
| `data[].displayName` | string | 사람이 읽기 좋은 이름 |
| `data[].pluralDisplayName` | string | 표시 이름의 복수형 |
| `data[].description` | string | 객체 유형 설명 |
| `data[].primaryKey` | string | 기본키 속성 이름 |
| `data[].properties` | object | 속성 이름 → 속성 정의 맵 |
| `data[].rid` | string | 리소스 식별자 |
| `nextPageToken` | string 또는 null | 다음 페이지 조회용 토큰 |

## 단일 객체 유형 조회

API 이름으로 단일 객체 유형을 조회합니다.

```
GET /api/v2/ontologies/{ontologyApiName}/objectTypes/{objectTypeApiName}
```

### 경로 파라미터

| 파라미터 | 타입 | 필수 | 설명 |
|:---------|:-----|:-----|:-----|
| `ontologyApiName` | string | 예 | 온톨로지의 API 이름 |
| `objectTypeApiName` | string | 예 | 객체 유형의 API 이름 (예: `Employee`) |

### 요청 예시

```bash
curl -X GET "http://localhost:8002/api/v2/ontologies/acme/objectTypes/Employee" \
  -H "Authorization: Bearer <token>"
```

### 에러 응답

| 상태 | 에러 코드 | 설명 |
|:-----|:---------|:-----|
| `404` | `ObjectTypeNotFound` | 지정한 객체 유형이 온톨로지에 존재하지 않음 |
| `404` | `OntologyNotFound` | 지정한 온톨로지가 존재하지 않음 |
| `401` | `Unauthorized` | 인증 토큰이 없거나 유효하지 않음 |
| `403` | `Forbidden` | 이 온톨로지에 대한 읽기 권한이 없음 |

### 속성 데이터 타입

`properties` 맵의 각 속성은 값 타입을 설명하는 `dataType` 객체를 포함합니다:

| 데이터 타입 | JSON 표현 | 예시 값 |
|:-----------|:----------|:--------|
| String | `{"type": "string"}` | `"Jane Doe"` |
| Integer | `{"type": "integer"}` | `42` |
| Long | `{"type": "long"}` | `9876543210` |
| Double | `{"type": "double"}` | `99999.99` |
| Boolean | `{"type": "boolean"}` | `true` |
| Date | `{"type": "date"}` | `"2024-01-15"` |
| Timestamp | `{"type": "timestamp"}` | `"2024-01-15T09:30:00Z"` |
| GeoPoint | `{"type": "geopoint"}` | `{"lat": 37.77, "lon": -122.42}` |
| Array | `{"type": "array", "itemType": {"type": "string"}}` | `["a", "b"]` |
