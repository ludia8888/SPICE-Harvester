Get Started in Minutes

From zero to a running platform in three steps. Full documentation covers every configuration option.

1

Install

Add SPICE Harvester to your project with a single command.

# Install the CLI and SDK npm install @spice/harvester-sdk
2

Configure

Connect to your data sources and define your ontology schema.

# spice.config.yaml ontology: namespace: production version: v2
3

Deploy

Ship to production with built-in observability and scaling.

# Deploy to your cluster spice deploy --env production # ✓ 6 services running
