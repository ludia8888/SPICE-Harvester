## 🔍 문서 표준

### 파일 명명 규칙
- 주요 가이드는 대문자 사용 (예: `DEPLOYMENT_GUIDE.md`)
- 언더스코어를 사용한 설명적인 이름
- 번역된 문서는 언어 접미사 포함 (예: `_KR.md`)

### 콘텐츠 구조
- 긴 문서에는 항상 목차 포함
- 명확한 섹션 헤더 사용
- 해당하는 경우 코드 예제 포함
- 코드 변경사항과 함께 문서 최신 상태 유지
