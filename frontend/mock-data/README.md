# Mock Data Fixtures

Synthetic CSV fixtures used by:

- `frontend/mock-server/` (dev-only mock API server)
- local demos / UI development

## Structure

- `*.csv`: raw-ish sample datasets
- `clean/*.csv`: “cleaned” versions of the same datasets

## Notes

- Do not put real customer/employee data in this folder.
- Keep IDs, emails, phone numbers, and names synthetic (`example.com`, dummy numbers, etc.).
